# PWA Icons

Add the following icon files to this directory for PWA support:

## Required Icons

| File | Size | Purpose |
|------|------|---------|
| `icon-72x72.png` | 72x72 | Small tile icon |
| `icon-192x192.png` | 192x192 | Medium tile icon |
| `icon-512x512.png` | 512x512 | Large tile icon |
| `badge-72x72.png` | 72x72 | Badge/shadow icon |

## Recommendations

- Use PNG format with transparency
- Match the app's theme color: `#E8B4A8`
- Icons should be simple and recognizable at small sizes
- You can use tools like [Favicon.io](https://favicon.io/) or [PWA Builder](https://www.pwabuilder.com/) to generate these
