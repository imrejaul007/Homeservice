# RFC Process

## Creating an RFC

1. Create `rfcs/0000-my-feature.md`
2. Fill template (Context, Decision, Consequences)
3. Share for review (2 weeks)
4. Address feedback
5. Merge after approval

## RFC Template

```markdown
# RFC-000: Feature Title

## Status: Draft|Review|Accepted|Rejected

## Context
Problem being solved.

## Decision
Chosen solution.

## Alternatives Considered
- Option A: pros/cons
- Option B: pros/cons

## Consequences
- Positive: benefits
- Negative: tradeoffs
```
