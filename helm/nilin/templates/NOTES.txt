# NILIN Helm Chart

## Installation

```bash
# Production
helm install nilin ./helm/nilin \
  -f ./helm/nilin/values-prod.yaml \
  --namespace nilin \
  --create-namespace

# Staging
helm install nilin-staging ./helm/nilin \
  -f ./helm/nilin/values-staging.yaml \
  --namespace nilin-staging \
  --create-namespace
```

## Required Secrets

Create a `values-secrets.yaml` file with:

```yaml
secrets:
  mongodbUri: "mongodb://user:password@host:27017/db"
  redisUrl: "redis://password@host:6379"
  jwtSecret: "your-secure-jwt-secret-256-bits"
  stripeSecret: "sk_live_xxx"
  resendApiKey: "re_xxx"
```

## Upgrades

```bash
helm upgrade nilin ./helm/nilin -f values-prod.yaml -f values-secrets.yaml --atomic
```

## Rollback

```bash
helm rollback nilin 1 --namespace nilin
```

## Uninstall

```bash
helm uninstall nilin --namespace nilin
```
